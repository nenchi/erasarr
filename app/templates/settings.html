Delete Directly from Emby/Jellyfin

Require All Users Watched
Only process an episode when every user covered by this rule has watched it — prevents removing episodes others haven't seen yet
Delete Older if Newer Watched
If a user has watched a newer episode (e.g. S02E14), automatically process older Sonarr-managed episodes on disk that are not in the watch history — assumes they were watched elsewhere. Episodes / Sonarr only. Works alongside "Require All Users Watched": explicit episodes still need all users to have watched them, while older episodes use the per-season safety check. Safety: a season is only touched if every targeted user has at least one recorded watch in that season.

When enabled, Keep Last N Episodes protects the N most recently watched episodes (by timestamp). When off (default), protects the N highest-numbered episodes in Sonarr.
